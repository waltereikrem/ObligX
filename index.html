Introduksjon

Velkommen til Walter Eikrem's Oblig X side. På denne siden vill du finne mange nyttige krep og triks, samt eksempler for deg som vill sette deg litt inn i webutvikling.

Innholdet vill variere fra basis og litt mer avansert CSS (stilkode. eks. farger og utforming), til html (elementer/koden man setter stilen på. aka Innholdet). Alt som er i bruk på denne nettsiden vill bli forklart på en simpel og enkel måte. Det skal være lett å forstå selv for de som er helt nybegynnere. Jeg håper at det å nylig ha lært mye nytt selv ang webutvikling vill hjelpe på forklaringene, i motsettning til sider alá CSS Tricks. som ofte kan ha litt komplisert forklaring for helt ferske utviklere.

Om det er noe du som nybegynner lurer på som kanskje ble forklart litt komplisert kan du sende mail til meg på waltere@hiof.no
